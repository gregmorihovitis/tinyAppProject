#LHL TinyApp Project
Used for shortening URLS of websites

##Greg Morihovitis

##Dependancies
-Node.js
-Express
-EJS
-bcrypt
-body-parser
-cookie-session

##Getting Started
-Install all dependencies using npm install
-Run the development web server by running expressServer.js file or npm start
